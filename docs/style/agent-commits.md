# Agent commit conventions (TASK-299)

Format: `<type>(<scope>): TASK-NNN — short description`

Types: feat, fix, docs, refactor, test, chore, revert, ci, build
Scope: chain, vm, api, db, agent, hud, ops, security, sdk, examples, runbooks

Body:
- 1-3 paragraphs explaining WHY, not WHAT
- Reference TASK-NNN if applicable
- Co-authored-by trailer for AI assistance
